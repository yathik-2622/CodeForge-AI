# @codeforge/cli

Command-line interface for CodeForge AI. Works on Windows, Linux, and macOS.

## Install

```bash
npm install -g @codeforge/cli
```

## Setup

```bash
codeforge config --server https://your-app.replit.app
codeforge status   # verify connection
```

## Usage

```bash
codeforge chat                          # interactive chat
codeforge chat "Fix this error: ..."   # one-shot question
codeforge search "React 19 features"   # web search
codeforge repos                         # list connected repos
codeforge sessions                      # list sessions
codeforge models                        # available AI models
```
